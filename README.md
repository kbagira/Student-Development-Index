# Student Development Index (SDI)

The **Student Development Index (SDI)** is a digital platform that allows students to register using their student ID, build a personal profile, and upload records of their extracurricular achievements such as volunteering, club involvement, projects, competitions, and more.  
Authorized verifiers review submissions and assign coins based on achievement value. The platform automatically generates student rankings based on accumulated coins.
